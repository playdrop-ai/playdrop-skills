# Assets And Generation

Use this when improving presentation with real assets or generating missing content quickly.

Catalog content can include:
- code samples
- 3D
- images
- video
- music
- SFX

Start with:
- `playdrop browse ...`
- `playdrop search <query>`
- `playdrop detail <ref> --json`
- `playdrop ai create <type> <prompt>`
- `playdrop ai jobs`

Decision rule:
- use catalog assets when you want a known style or faster integration
- use AI generation when you need custom variations or missing pieces
- inspect licensing, fit, and production quality before integrating
