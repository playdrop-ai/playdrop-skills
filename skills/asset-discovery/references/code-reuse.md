# Code Reuse

Use this when reusing published Playdrop app code.

## Workflow

- search for the closest reference
- inspect details before downloading
- remix or download locally only after confirming fit
- extract the specific pattern needed instead of copying the whole project blindly
- document what was reused and why

## Preferred targets

- starter kits
- demos with clear one-feature implementations
- projects whose runtime constraints match the new game
