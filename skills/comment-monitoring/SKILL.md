# Comment Monitoring

Use this skill to turn comments into useful follow-up work instead of treating them as passive noise.

## Triage flow

1. review recent comments
2. classify each one
3. answer direct questions
4. turn useful reports into concrete fixes or backlog items
5. ignore or remove obvious spam through the right channel

## Shared references

- `comment-triage.md`
- `live-ops.md`

## Handoff

- bug fixes or polish work -> `game-improvement`
- feature requests or roadmap decisions -> `game-planning`
- listing confusion or weak public positioning -> `store-listing`
