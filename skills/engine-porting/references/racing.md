# Racing

Use this for kart, racing, driving, or vehicle games.

Start with:
- `playdrop detail playdrop/app/starter-kit-racing --json`
- `playdrop search racing --kind app --app-type demo --json`
- `playdrop browse --kind app --app-type template --json`
- `playdrop search kart --kind asset --json`
- `playdrop search racing --kind asset-pack --json`

Default rule:
- inspect `playdrop/app/starter-kit-racing` early
- use it as the default reference for controls, physics, camera, HUD, assets, and mobile controls unless a better fit exists

Good searches:
- apps and demos : `playdrop search racing --kind app --app-type demo --json`, `playdrop search kart --kind app --json`
- templates : `playdrop browse --kind app --app-type template --json`
- 3D assets : `playdrop search kart --kind asset --json`, `playdrop search vehicle --kind asset --json`, `playdrop search track --kind asset --json`
- asset packs : `playdrop search racing --kind asset-pack --json`, `playdrop search low poly --kind asset-pack --json`

Suggested result:
- `Starter Kit Racing Demo` is a strong first reference when you want both gameplay code and simple 3D racing assets to borrow from or align with

Compare:
- driving feel
- camera and readability
- HUD and lap flow
- mobile controls
- asset style and track dressing

If you choose not to use the starter-kit reference, say why.
