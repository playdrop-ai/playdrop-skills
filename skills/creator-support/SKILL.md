# Creator Support

Use this skill when the creator is blocked and the main value is fast, practical unblocking.

## Escalation order

1. check live docs with `playdrop documentation browse`
2. check `playdrop help` and `playdrop getting-started`
3. submit actionable product feedback with `playdrop feedback send`
4. point the creator to the public community channel if needed

## Rules

- prefer the simplest next step
- use public documentation and public workflows only
- if the real need is a specialist workflow, hand off to the matching skill

## Handoff map

- new idea, scope, or platform fit questions -> `game-planning`
- cut a plan down to a smaller v1 -> `scope-control`
- gameplay mockups, HUD framing, or visual build-target review -> `gameplay-mockups`
- finding starter kits, assets, or remix candidates -> `asset-discovery`
- existing project setup, migration, or metadata update questions -> `project-updates`
- local setup, validation, or multiplayer checks -> `dev-testing`
- pre-publish feel or session-quality review -> `gameplay-review`
- runtime quality, polish, or iteration work -> `game-improvement`
- publishing, capture, listing media, or tags -> `store-listing`
- post-launch comments and maintenance -> `comment-monitoring`
- services, auth, saves, realtime, monetization, or AI features -> `service-integration`
- engine migration or porting -> `engine-porting`
