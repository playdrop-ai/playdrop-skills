# Roadmap Phases

Use these phases as the default public roadmap structure.

## Phase 1: Foundation

- project boots locally
- one core loop works
- placeholder art is acceptable

## Phase 2: Vertical Slice

- one representative end-to-end slice
- target visual language established
- controls and feel are credible

## Phase 3: Full Content

- core content set complete
- systems integrated
- validation passes

## Phase 4: Polish And Release

- listing assets ready
- performance issues addressed
- comments and marketing plan prepared
