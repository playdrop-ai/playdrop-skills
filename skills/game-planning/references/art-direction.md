# Art Direction

Use this when planning visuals or choosing assets.

## Approach

- inspect 2 to 3 strong Playdrop references before generating anything
- define the visual promise in a short phrase
- choose one dominant palette and silhouette language
- decide what comes from catalog assets versus custom generation
- make icon, hero, screenshots, and runtime art feel like the same game

## Avoid

- mixing unrelated styles
- using hero art that does not match the runtime
- shipping a raw crop as the final icon
