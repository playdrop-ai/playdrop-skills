# CLI Map

Use this when you need the right Playdrop command family fast.

- auth : `playdrop auth ...`
- docs : `playdrop documentation ...`
- discovery : `playdrop browse ...`, `playdrop search ...`, `playdrop detail ...`
- AI generation : `playdrop ai create ...`, `playdrop ai jobs`
- existing project : `playdrop project init .`, `playdrop project validate .`, `playdrop project dev ...`
- new app : `playdrop browse --kind app --app-type template --json`, `playdrop project create app <name> --template <ref>`
- remix app : `playdrop project create app <name> --remix <ref>`
- capture and publish : `playdrop project capture ...`, `playdrop project publish .`
- manage creations : `playdrop creations browse ...`, `playdrop creations apps update ...`
- comments and releases : `playdrop comments ...`, `playdrop versions ...`

Then use CLI help for exact flags and current options.
