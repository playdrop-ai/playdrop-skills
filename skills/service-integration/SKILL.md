# Service Integration

Use this skill when the request is about Playdrop platform capabilities rather than listing or pure gameplay polish.

## Workflow

1. read the relevant live public docs first
2. pick only the services that genuinely help the project
3. integrate in the simplest credible way
4. validate with the dev flow and test accounts
